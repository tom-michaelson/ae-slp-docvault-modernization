# Awa.Client SDK

A .NET client SDK for Awa services and APIs.

## Installation

Install the package via NuGet Package Manager:

```bash
dotnet add package Awa.Client
```

Or via Package Manager Console:

```powershell
Install-Package Awa.Client
```

## Requirements

- .NET 8.0 or later
- C# 8.0 or later

## Usage

```csharp
using Awa.Client;

// Your code here
```
