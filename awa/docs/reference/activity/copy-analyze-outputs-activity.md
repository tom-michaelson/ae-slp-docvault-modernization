# `copy_analyze_outputs_activity`

Copy analyze mode outputs from isolated environment to the original repository.

This activity copies any files generated by the agent during analyze mode
from the output directory in the isolated environment to the output directory
in the original repository.

## Parameters

| Name               | Type                           | Description                                                                       |
| :----------------- | :----------------------------- | :-------------------------------------------------------------------------------- |
| `environment_info` | `IsolatedAgentEnvironmentInfo` | Information about the isolated environment containing outputs                     |
| `output_directory` | `str`                          | Directory name for agent outputs in analyze mode (defaults to "awa-agent-output") |

## Returns

| Type                             | Description                       |
| :------------------------------- | :-------------------------------- |
| `IsolatedAgentEnvironmentResult` | Result object with success status |

## Usage

::: code-group

```python [Python]
from temporalio import workflow
from datetime import timedelta
from awa.core.activities.agent_modes.models.isolated_agent_models import IsolatedAgentEnvironmentInfo

@workflow.defn
class MyWorkflow:
    @workflow.run
    async def run(self, environment_info: IsolatedAgentEnvironmentInfo, output_directory: str = "awa-agent-output"):
        return await workflow.execute_activity(
            "copy_analyze_outputs_activity",
            args=[environment_info, output_directory],
            task_queue="awa_default",
            start_to_close_timeout=timedelta(minutes=5)
        )
```

```go [Go]
import (
	"time"

)

func MyWorkflow(ctx workflow.Context, environmentInfo interface{}, outputDirectory string) (interface{}, error) {
	ao := workflow.ActivityOptions{
		TaskQueue:           "awa_default",
		StartToCloseTimeout: time.Minute * 5,
	}
	ctx = workflow.WithActivityOptions(ctx, ao)

	var result interface{}
	err := workflow.ExecuteActivity(ctx, "copy_analyze_outputs_activity", environmentInfo, outputDirectory).Get(ctx, &result)
	return result, err
}
```
