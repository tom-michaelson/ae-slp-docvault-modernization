# LLM Configuration

LLMs are configured in `config.yaml`.

See our [Reference](/reference/configuration/application) for a complete list of configuration options.

## Supported Providers

- [Anthropic](/configuration/llms/anthropic)
- [AWS Bedrock](/configuration/llms/aws-bedrock)
- [Azure OpenAI](/configuration/llms/azure-openai)
- [Google Vertex AI](/configuration/llms/google-vertex)
- [Github Copilot](/configuration/llms/github-copilot)
- [OpenAI](/configuration/llms/openai)
- [LiteLLM](/configuration/llms/lite-llm) (just about any provider can be used via the LiteLLM proxy, but Docker is required)
