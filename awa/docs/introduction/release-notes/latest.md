## AWA CLI

Use the AWA CLI to start all services. See [CLI Usage](/usage/cli) for details.

## AWA MCP Server

Run AWA as an MCP server to easily integrate with your agentic IDE (Github Copilot, Cursor, etc.). See [MCP Usage](/usage/mcp) for details.

## Reusable Workflows and Activities

AWA comes bundled with a large set of reusable Temporal workflows and activities to accelerate workflow development. These workflows and activities can be used as building blocks to create your own agentic workflows.

See our reference for [Workflows](/reference/workflow/) and [Activities](/reference/activity/) for details on what is available and how to use them.
