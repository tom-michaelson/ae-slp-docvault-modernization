# Roadmap

## In Progress

Our core team is actively working on the following features.

### Human-in-the-Loop UI

We are enhancing the AWA UI to support rich human-in-the-loop (HITL) interactions. This will include UI components and flows for user input, as well as related reusable child workflows and activities.

### Advanced Document Handling

Reusable child workflows and activities for parsing a wide variety of document types and chunking large documents.

### Cloud Deployment Blueprints

IaC deployment blueprints for AWS, Azure, and GCP to enable client teams to deploy quickly.

### Agent Isolation + Parallelization

Reusable child workflows and activities to support running CLI agents in container-isolated environments (using Dagger!). Recipes demonstrating heavy agent parallelization for the most complex use cases.
