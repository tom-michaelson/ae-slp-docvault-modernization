# Starter Projects

A core benefit of AWA (and Temporal) is that you can build your client application in any language supported by Temporal's SDKs (Python TypeScript, C#, Java, Go, or PHP). The example starter projects demonstrate how to get started with AWA in each of these languages.

The starter projects are located in the `cookbook/client-workers/` directory:

- [Dotnet (C#)](https://bitbucket.org/slalom-consulting/agentic-workflow-accelerator/src/main/cookbook/client-workers/dotnet/README.md) - See `cookbook/client-workers/dotnet/`
- [Java](https://bitbucket.org/slalom-consulting/agentic-workflow-accelerator/src/main/cookbook/client-workers/java/README.md) - See `cookbook/client-workers/java/`
- **TypeScript** - Coming soon!
