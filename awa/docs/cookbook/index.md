# Cookbook

The AWA Cookbook contains examples, tutorials, and starter projects that demonstrate how to build agentic workflows. All cookbook content is included in this repository under the `cookbook/` directory. Some recipes are standalone, while others are part of larger tutorial series. See the listings below for more details.

## Tutorials

Each tutorial is a step-by-step guide to building workflows. They deliberately demonstrate a wide set of features of AWA and related tooling, as well as general agentic workflow patterns.

- [AWA 101](/cookbook/tutorials/awa-101/) - Learn the basics of AWA workflows while building a workflow that extracts technical requirements from code.
- [AWA 201](/cookbook/tutorials/awa-201/) - Learn how to use advanced AWA features while building a workflow that generates a documentation site from a codebase.

## Recipes

Each recipe includes:

- Complete code examples
- Step-by-step usage instructions
- Command-line execution examples
- Best practices and tips

Choose an example that matches your use case and follow the instructions to get started with AWA workflows!
