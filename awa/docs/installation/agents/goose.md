# Agent - Goose

This document provides instructions on how to install and configure the Goose SWE Agent for use with AWA.

Goose is a CLI and Desktop agent developed by Block. For complete details, refer to the official [Goose Installation Documentation](https://block.github.io/goose/docs/getting-started/installation/).

## Installation

See the [official docs](https://github.com/block/goose) for information about how to install Goose on your system.

## Observability

Goose can be configured to send traces to Langfuse. See the [Langfuse](https://langfuse.com/docs/integrations/goose) docs for more details.
