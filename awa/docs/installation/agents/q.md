# Agent - Amazon Q

This document provides instructions on how to install and configure Amazon Q for use with AWA.

For detailed information, refer to the official [Amazon Q Developer User Guide](https://docs.aws.amazon.com/amazonq/latest/qdeveloper-ug/what-is-q-developer.html).

## Installation

See the official documentation for information about how to install Amazon Q on your system:

*   [Installing Amazon Q for command line](https://docs.aws.amazon.com/amazonq/latest/qdeveloper-ug/command-line-installing.html)

## Configuration

Using Amazon Q requires an AWS Builder ID or an IAM Identity Center login. Follow the instructions in the official documentation to log in after installation.

You can log in by running:
```bash
q login
```

You will be prompted to choose between a free tier with AWS Builder ID or a pro tier with IAM Identity Center.
