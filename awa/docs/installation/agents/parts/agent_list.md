Currently, AWA supports the following Agents:

- PydanticAI (Default Agent) - Built into AWA (no installation required)
- [Claude Code](/installation/agents/claude) \*
- [Gemini CLI](/installation/agents/gemini)
- [GitHub Copilot CLI](/installation/agents/github-copilot-cli) \*
- [Goose](/installation/agents/goose)
- [OpenAI Codex](/installation/agents/codex)
- [OpenCode](/installation/agents/opencode) \*
- [Amazon Q](/installation/agents/q)

\* These agents work on Windows without requiring WSL (Windows Subsystem for Linux).
