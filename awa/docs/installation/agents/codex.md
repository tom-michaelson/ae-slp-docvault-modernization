# Agent - OpenAI Codex

This document provides instructions on how to install and configure the Codex SWE Agent for use with AWA.

Codex is a lightweight coding agent from OpenAI that runs in your terminal. For more details, refer to the official [Codex GitHub repository](https://github.com/openai/codex).

## Experimental Technology Disclaimer

**Important:** Codex CLI is an experimental project under active development. It is not yet stable, may contain bugs, incomplete features, or undergo breaking changes. Use with caution.

## Installation

See the [official docs](https://github.com/openai/codex) for information about how to install Codex on your system.
