:::danger SDK Usage in Client Environments
The AWA SDK is published to internal Slalom AWS CodeArtifact package repositories which are **not directly accessible** from client environments. This means that if you intend to build with the AWA SDK in a client environment, you will need to download the SDK package manually from your Slalom hardware and copy it over to your client environment, referencing it as a local package. This is a limitation enforced by Slalom IT.
:::
